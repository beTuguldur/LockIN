# LockIn 👋

LockIn is a cross-platform productivity app designed to help you eliminate distractions and stay focused on what matters. Built with modern mobile technologies, it enables structured deep-work sessions and tracks your consistency over time.

🚀 Features
⏱️ Focus / Pomodoro-style sessions
🚫 Distraction reduction
📊 Progress tracking
📱 Cross-platform (iOS & Android)

🛠️ Tech Stack
   * React Native
   * Expo
   * JavaScript / TypeScript
   * Async Storage (or whatever you use)

npm install
npx expo start

📱 Run on
   * iOS Simulator
   * Android Emulator
   * Expo Go (QR scan)

Future Improvements
   *User authentication
   *Cloud sync
   *Advanced analytics
   *Custom focus modes
